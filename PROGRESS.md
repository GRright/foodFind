## 技术栈
- **前端框架**：uni-app + Vue 3
- **开发语言**：JavaScript
- **样式**：SCSS
- **部署平台**：微信小程序

## 项目结构
```
foodFind/
├── src/
│   ├── pages/           # 页面目录
│   ├── utils/           # 工具模块
│   ├── static/          # 静态资源
│   └── pages.json       # 页面配置
├── cloudfunctions/      # 云函数（已清理大部分）
└── PROGRESS.md         # 进度文档
```

## 核心功能
1. **智能菜单生成**：根据偏好和人数生成一周菜单
2. **营养均衡算法**：自动搭配荤素，确保营养均衡
3. **双人互动**：分享菜单、双向确认、打卡监督
4. **购物清单**：自动从菜单提取食材，智能分类
5. **家庭群组**：2-8 人家庭，共享购物清单，打卡看板，周报统计
6. **健康标签**：20 种标签过滤菜谱，适配特殊饮食需求

## 设计理念
- **简洁美观**：北欧极简设计风格
- **流畅体验**：性能优化，快速加载
- **智能决策**：算法推荐，减少选择困难
- **社交互动**：双人/多人互动，增加趣味性
