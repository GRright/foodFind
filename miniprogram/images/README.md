# TabBar 图标

请在此目录下放入以下图标文件（建议尺寸 81px × 81px）：

## 需要的文件：
- `tab-home.png` - 首页（未选中）
- `tab-home-active.png` - 首页（选中）
- `tab-menu.png` - 菜单（未选中）
- `tab-menu-active.png` - 菜单（选中）
- `tab-movies.png` - 影视（未选中）
- `tab-movies-active.png` - 影视（选中）
- `tab-profile.png` - 我的（未选中）
- `tab-profile-active.png` - 我的（选中）

## 获取图标的方式：
1. 从 iconfont、iconpark 等图标库下载
2. 使用设计工具（Figma、Sketch）创建
3. 从现成的微信小程序模板中获取
